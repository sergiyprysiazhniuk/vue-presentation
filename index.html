			data: {
			a: 1
			},

			console.log('a is: ' + this.a)

			// => "a is: 1"

Application as a tree of reusable self-contained components

Single file component

  • Single file .vue where template, javascript and css are encapsulated
  • Build tool like webpack ar browserify is needed
  • vue-loader for and vuefy for browserify
  • Complete syntax highlighting
  • CommonJS modules
  • Component scoped

Image

Vue Router

  • Nested route/view mapping
  • Modular, component-based router configuration
  • Route params, query, wildcards
  • Navigation Guards
  • View transition effects powered by Vue.js' transition system
  • Fine-grained navigation control
  • Links with automatic active CSS classes
  • HTML5 history mode or hash mode, with auto-fallback in IE9
  • Customizable Scroll Behavior

Server-Side Rendering

Allows you to render your apps on the server, then hydrate them with client side reactivity after the page loads, greatly increasing the responsiveness and improving the load time of your pages

Nuxt.js

Nuxt.js is a higher-level framework built on top of the Vue ecosystem which provides an extremely streamlined development experience for writing universal Vue applications

Nuxt.js Features

  • Write Vue Files (*.vue)
  • Automatic Code Splitting
  • Server-Side Rendering
  • Powerful Routing System with Asynchronous Data
  • ES6/ES7 Transpilation
  • Hot module replacement in Development
  • Pre-processor: Sass, Less, Stylus, etc.
  • HTTP/2 push headers ready
  • Extending with Modular architecture
